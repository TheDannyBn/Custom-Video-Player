Description
This project demonstrates how to use the HTML5 Video API to create an interactive video player. The project includes features like play, pause, stop, volume control, and fullscreen mode, providing a comprehensive guide to manipulating video elements using JavaScript.

Features
Play/Pause: Start and pause the video.
Stop: Stop the video and reset the playback position.
Seek: Jump to different parts of the video.
Volume Control: Adjust the volume of the video.
